# 分散系

将一种或多种物质分散到另一种物质中形成的体系。

根据分散质粒子直径 $d$，可分为以下三种：

- [溶液](/definitions/solution)—— $d < 1\text{ nm}$
- [胶体](/definitions/colloid)—— $1\text{ nm} \le d \le 100\text{ nm}$
- [浊液](/definitions/turbid-liquid)—— $d > 100\text{ nm}$
