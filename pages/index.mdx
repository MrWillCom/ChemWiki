## 规范

### 文本

对于一般的中文文案排版，遵循 [sparanoid/chinese-copywriting-guidelines](https://github.com/sparanoid/chinese-copywriting-guidelines) 的方案。

### 路径

使用全英文路径，将空格改为 `-`，并全部小写。如 Sulfur Elementary Substance 应写为 `sulfur-elementary-substance`。

### 公式（LaTeX）

数学表达式、化学方程式等公式，应使用 LaTeX 表示，如：

$\ce{Cl2}$ 与 $\rm SO_2$ $1:1$ 混合，失去漂白作用。

$$
\ce{SO2 + Cl2 + 2H2O = H2SO4 + 2HCl}
$$


### 组件

得益于 [Nextra](https://nextra.site/) 的内建组件，你可以使用它们代表更丰富的含义：

如 `<Callout />` 用于强调一些内容：

<Callout type="warning">
  该反应是中学化学中唯一一个同时生成烟和雾的反应。
</Callout>

`<Callout />` 有多种类型，请参阅：[Callout Component - Nextra](https://nextra.site/docs/docs-theme/built-ins/callout)。

另外还有 `<Tabs />` 和 `<Tabs.Tab />` ，请参阅：[Tab Component - Nextra](https://nextra.site/docs/docs-theme/built-ins/tabs)。

## 开发

### 安装依赖

```sh
pnpm install
```

### 启动开发服务器

```sh
pnpm dev
```

### 为生产环境构建

```sh
pnpm build
```

### 在生产环境启动

```sh
pnpm start
```
