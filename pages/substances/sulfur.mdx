# 硫

| Sulfur       | $\ce{S}$ |
| ------------ | -------- |
| 序数         | 16       |
| 族           | 16 · VIA |
| 相对原子质量 | 32       |
| 核外电子排布 | 2, 8, 6  |

硫在自然界中，以游离态的形式存在于火山口附近或地壳的岩层里。以化合态的硫化物和硫酸盐的形式存在。与金属元素形成的化合物通常以硫化物的形式存在，如硫铁矿（也称“黄铁矿”，$\ce{FeS2}$）、黄铜矿（$\ce{CuFeS2}$）。在地表附近，硫化物转化为硫酸盐的形式。另外，有机物中也有硫的出现。
