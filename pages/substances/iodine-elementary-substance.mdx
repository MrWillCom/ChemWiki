# 碘（单质）

一种紫黑色[卤素](/substances/halogen)单质固体。

## 物理性质

| 性质          |                         |
| ------------- | ----------------------- |
| 颜色          | 紫黑色                  |
| 密度 ($\rho$) | $4.63\text{ g/mL}$      |
| 溶解度        | $0.029\text{ g/100g水}$ |
| 熔点          | $113.6\celsius$         |
| 沸点          | $185.2\celsius$         |
| 金属光泽      | 是                      |

难溶于水，但溶于苯、汽油、$\rm CCl_4$、酒精等有机溶剂。溶于酒精则制得碘酒，一种棕黄色溶液。

易升华，其蒸汽——紫红色碘蒸汽——易凝华。

储存于棕色广口瓶。
